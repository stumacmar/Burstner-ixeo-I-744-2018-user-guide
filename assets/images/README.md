# Image Assets\n\nThis directory contains images for the Bürstner Ixeo I 744 interactive manual.\n\n## Structure\n- **heating/** - Alde heating system images\n- **power/** - Solar, battery, and electrical images\n- **cameras/** - 360° camera and reversing camera images\n- **media/** - TV and entertainment system images\n- **beds/** - Drop-down and pull-down bed images\n- **comfort/** - Blinds, flyscreens, etc.\n- **exterior/** - Awning, heat shield, outdoor features\n- **safety/** - Alarm, tracker images\n- **water/** - Fresh water, outdoor shower\n- **toilet/** - Cassette toilet images\n\n## Adding Images\n\nReplace placeholder filenames with actual photos. Supported formats: JPG, PNG, WebP.\n\nExample: Replace 'alde-control-panel.jpg' with your own photo of the same name.
