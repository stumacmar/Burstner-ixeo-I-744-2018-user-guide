## Photo Tips

1. **Good lighting** - Take photos in daylight or with good interior lighting
2. **Clear focus** - Ensure controls and labels are readable
3. **Consistent size** - Aim for at least 800x600 pixels
4. **Landscape orientation** - Works best for the gallery layout
5. **Show context** - Include surrounding area to help locate items

## Recommended Image Sizes

- Gallery thumbnails: 400x300 pixels minimum
- Full-size viewing: 1200x900 pixels recommended
- File format: JPEG (.jpg) for photos
- File size: Under 500KB per image for fast loading
